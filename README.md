# desafio-android
Versão alternativa do projeto android.

Retornando aos estudos de HTML e CSS com um projeto inicial que aborda propriedades interessantes de ambas as ferramentas.
A página conta a história do mascote do Android. O projeto possui alguma estrutura semântica (dentro da proposta do exercício) bem como algumas características interessantes de estilização, como gradientes e responsividade. 

A versão final do projeto pode ser acessada <a href="https://luizfernandolpp.github.io/desafio-android/" target="_blank">aqui</a>.
